# Output Structure

## Module meanings

### 1. 镜头身份
Defines:
- shot id
- duration
- type
- plan
- tier
- tempo

### 2. 挂载
Only entities:
- role
- scene
- prop

No action phrases here.

### 3. 镜头命题
One stable line:
- what this shot fundamentally does
- what dramatic function it serves

### 4. 场面机制
Must include:
- main event
- relation change
- result state

### 5. 空间机制
Must include:
- split trigger
- foreground / midground / background
- center object
- occlusion

### 6. 镜头执行
Must include:
- camera position
- camera facing
- character facing when applicable
- lens
- depth of field
- focus priority

### 7. 时间推进
At least 3 beats:
- 事件成立
- 信息压入
- 结果钉住

### 8. 声画规则
Must include:
- visible speaker
- hidden speaker
- primary sound
- secondary sound
- what the sound suppresses

### 9. 结果锚定
Must include:
- Must-Show
- hard nails
- emotion nail
- narrative nail
- relay object / handoff

## Tier differences

### LITE
- shortest stable version
- remove repetition first
- remove noisy fragments first
- preserve only strong anchors

### STD
- balanced expansion
- keeps full structure

### PRO
- highest control density
- allows more detail in continuity, sound pressure, steady-frame logic
